\doxysubsection{Detailed Description}
holds a pre-\/trained Open Neural Network e\+Xchange model and provides some simple functionality to manipulate input and output data 

ML.\+NET complicates this process by requiring all data entering ML.\+NET to have \char`\"{}\+IData\+View\char`\"{} interface implemented within it\textquotesingle{}s own user defined data class. This makes it quite difficult to have a flexible data class which can handle arbitrary input size depending on the model fed to it. Right now, the primitive\+\_\+type variable must match both the input and output layer primitive data type.
