\doxysubsection{Detailed Description}
Creates an object that runs on it\textquotesingle{}s own thread to manage data collection at reliable time intervals 

Note that the thread spins when not logging to update its timer and therefore clogs cpu usage. This makes data sample rate as consistent as possible on a windows OS
