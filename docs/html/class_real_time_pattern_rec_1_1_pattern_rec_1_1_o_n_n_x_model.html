Detailed Description

holds a pre-trained Open Neural Network eXchange model and provides some simple functionality to manipulate input and output data

ML.NET complicates this process by requiring all data entering ML.NET to have "IDataView" interface implemented within it's own user defined data class. This makes it quite difficult to have a flexible data class which can handle arbitrary input size depending on the model fed to it. Right now, the primitive_type variable must match both the input and output layer primitive data type.
